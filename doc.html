Postup instalace

Požadované nástroje:

  • Docker + Docker Compose
  • Java 17
  • Maven 3.9+
  • Node.js 20.19+

Spustenie databáze:

  • docker compose up -d v root adresári projektu
  • Backend pri spustení automaticky vkladá demo dáta, ak chýbajú

Spustenie backendu:

  • mvn liberty:dev v priečinku hotel-backend
  • Backend pobeží tu

Spustenie frontendu:

  • cd hotel-frontend
  • npm install
  • npm run dev
  • Frontend typicky beží tu
